# Undefinedd

Two D's for a double dose of undefined

## Install

```bash
npm install undefinedd
```

## Usage

```bash
const undefinedd = require("undefinedd");

console.log(undefinedd); // print "undefined"
```

## Test

```bash
npm run test
```
